# Simple Music Player 使用说明

## 版本 v1.0.1 - 修复乱序播放bug

### 启动程序
```bash
./build/smp
```

### 基本操作

#### 播放界面
- **空格**: 播放/暂停
- **← →**: 上一首/下一首
- **M**: 主菜单
- **A**: 添加到歌单
- **H**: 帮助
- **Q**: 退出

#### 菜单导航
- **↑ ↓**: 上下移动 (在当前页内)
- **PgUp/PgDn**: 翻页
- **Enter**: 选择
- **ESC**: 返回上级
- **H**: 显示帮助

### 歌单系统

#### 创建歌单
1. 主菜单 → 歌单管理器
2. 按 **C** 创建歌单
3. 输入歌单名称
4. 选择创建方式:
   - 从目录添加: 输入目录路径，自动添加所有 MP3/FLAC 文件
   - 创建空歌单: 创建一个空的歌单

#### 管理歌单
在歌单管理器界面:
- **Enter**: 浏览歌单
- **D**: 删除歌单
- **R**: 重命名歌单

#### 浏览歌单
在歌单浏览界面:
- **Enter**: 播放选中的歌曲
- **D**: 删除选中的歌曲
- **S**: 排序歌单
- **ESC**: 返回歌单管理器

#### 添加歌曲到歌单
在播放界面:
- 按 **A** 打开"添加到歌单"菜单
- 选择目标歌单
- 当前播放的歌曲会被添加到选中的歌单 (自动查重)

#### 排序歌单
1. 在歌单浏览界面按 **S**
2. 选择排序方式:
   - 按歌名排序
   - 按歌手排序
   - 按专辑排序
   - 按文件名排序
   - 按修改时间排序
3. 选择排序顺序: A(升序) 或 D(降序)

### 设置

#### 播放模式
- 主菜单 → 设置 → 播放模式
- 顺序播放: 按列表顺序播放
- 乱序播放: 随机播放歌单中的歌曲

### 数据存储

程序数据存储在 `~/.config/simple_music_player/` 目录:

```
~/.config/simple_music_player/
├── config.json              # 主配置文件
└── song_lists/              # 歌单目录
    ├── playlist_0.json      # 歌单0
    ├── playlist_1.json      # 歌单1
    └── ...
```

#### 配置文件格式
```json
{
  "play_mode": "sequential",  // 或 "shuffle"
  "current_playlist_index": 0,
  "current_song_index": 0,
  "playlists_meta": [
    {
      "index": 0,
      "name": "默认歌单",
      "created_time": 1741348800,
      "modified_time": 1741348800
    }
  ]
}
```

#### 歌单文件格式
```json
{
  "name": "歌单名称",
  "created_time": 1741348800,
  "modified_time": 1741348800,
  "songs": [
    {
      "path": "/path/to/song.mp3",
      "title": "歌曲标题",
      "artist": "艺术家",
      "album": "专辑",
      "modified_time": 1741348800
    }
  ]
}
```

### 构建与安装

```bash
# 构建
bash ./build.sh

# 安装 (生成 DEB 包)
cd build
cpack

# 安装 DEB 包
sudo dpkg -i simple-music-player-0.2.0-Linux.deb
```

### 依赖库
- SDL2 / SDL2_mixer (音频播放)
- TagLib (元数据解析)
- NcursesW (终端界面)
- nlohmann_json (配置存储)
- CMake (构建系统)

### 故障排除

#### 1. 无法播放音乐
- 检查 SDL2_mixer 是否正确安装
- 检查音频文件格式是否支持 (MP3, FLAC)

#### 2. 无法读取元数据
- 检查 TagLib 是否正确安装
- 检查音频文件是否有正确的标签信息

#### 3. 界面显示异常
- 确保终端支持 UTF-8 编码
- 检查终端尺寸是否足够大

#### 4. 歌单文件损坏
- 备份 `~/.config/simple_music_player/song_lists/` 目录
- 删除损坏的歌单文件，程序会自动创建默认歌单
