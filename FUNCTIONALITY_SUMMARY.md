# 功能修改总结

## 已完成的功能修改

### 1. 按键提示简化
- **问题**: 原界面显示所有按键提示
- **修改**: 所有界面只显示 `[H]帮助` 提示
- **实现**:
  - `src/main.cpp`: 播放界面底部提示改为 `[H]帮助`
  - `src/UIHelpers.cpp`: 菜单界面底部提示改为 `[H]帮助`
  - 移除了所有其他按键提示

### 2. ESC 键功能改为 Q 键
- **问题**: 使用 ESC 键返回上级
- **修改**: 使用 Q 键，在播放界面退出程序，在其他界面返回上级
- **实现**:
  - 移除了所有 `ch == 27` (ESC) 的处理
  - 在各个输入处理函数中添加 Q 键处理:
    - 播放界面: `ctrl.stop()` 退出程序
    - 主菜单: 返回播放界面
    - 歌单管理器: 返回主菜单
    - 歌单浏览: 返回歌单管理器
    - 创建歌单: 返回歌单管理器
    - 添加到歌单: 返回播放界面
    - 排序菜单: 返回歌单浏览
    - 设置: 返回主菜单
    - 播放模式: 返回设置

### 3. 歌单功能实现
- **问题**: 歌单的增加、删除、编辑、排序功能需要实现
- **修改**: 完整实现了歌单管理系统
- **实现**:

#### 歌单管理 (`AppController`)
- `createPlaylist()`: 创建歌单
- `deletePlaylist()`: 删除歌单（同时删除文件）
- `renamePlaylist()`: 重命名歌单
- `addSongsFromDirectory()`: 从目录添加歌曲到歌单
- `addCurrentSongToPlaylist()`: 添加当前播放歌曲到歌单（自动查重）
- `removeSongFromPlaylist()`: 从歌单删除歌曲
- `sortPlaylist()`: 排序歌单（支持5种排序方式，正序/倒序）

#### 文件存储
- 每个歌单保存为单独文件: `~/.config/simple_music_player/song_lists/playlist_*.json`
- 配置文件只保存歌单元信息
- 支持歌单文件的创建、保存、加载、删除和重命名

#### 界面操作
- **创建歌单**: 歌单管理器按 `C` → 输入名称 → 选择创建方式
- **删除歌单**: 歌单管理器按 `D`
- **重命名歌单**: 歌单管理器按 `R` → 输入新名称
- **删除歌曲**: 歌单浏览按 `D`
- **排序歌单**: 歌单浏览按 `S` → 选择排序方式 → 选择排序顺序

### 4. 帮助系统改进
- **问题**: 帮助信息显示所有界面的功能
- **修改**: 每个界面显示特定的帮助信息
- **实现**:

#### 帮助信息定义 (`UIHelpers.cpp`)
- `PLAYING_HELP`: 播放界面帮助
- `MAIN_MENU_HELP`: 主菜单帮助
- `PLAYLIST_MANAGER_HELP`: 歌单管理器帮助
- `PLAYLIST_VIEW_HELP`: 歌单浏览帮助
- `PLAYLIST_EDIT_HELP`: 创建歌单帮助
- `ADD_TO_PLAYLIST_HELP`: 添加到歌单帮助
- `SORT_MENU_HELP`: 排序方式帮助
- `SETTINGS_HELP`: 设置帮助
- `PLAY_MODE_HELP`: 播放模式帮助

#### 帮助系统逻辑
- 按 `H` 进入帮助界面，记录当前状态
- 根据进入前的状态显示相应的帮助信息
- 按任意键返回之前的状态
